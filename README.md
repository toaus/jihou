# jihou
簡易的な時報アプリ。  
NTTの117ライクで、時刻の読み上げ機能もあります。  
四国めたんちゃんが読み上げます。  
ちなみにアクセントは本物に似せています。結構こだわりポイント。  

## 使用・ビルド前準備
python3が必要。  
また、pygameを予めpipでインストールしておくこと。
そして、事前にNTPで時刻同期しておくこと。

## 用法
jihou.pyを実行する。  
ウィンドウを閉じるか、Ctrl+Cを押すと終了。
